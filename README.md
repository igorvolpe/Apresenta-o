# 💝 Nossa Jornada: 4 Meses de Nós

Este é um projeto web interativo e personalizado, desenvolvido para celebrar os primeiros 4 meses de um relacionamento especial. O site combina narrativa, planos futuros e interatividade para simbolizar nossa conexão.

---

## 🚀 Funcionalidades Principais

* **Linha do Tempo Interativa:** Navegação fluida entre os marcos da nossa história.
* **Contador de Batimentos:** Um script que calcula em tempo real quantas vezes meu coração bateu por você desde **22/11/2024**.
* **Caça-Palavras do Amor:** Um mini-jogo personalizado onde é necessário encontrar palavras-chave (**GONÇALVES, AMOR, ITAPIPOCA, ITÁLIA, VIDA**) para liberar a mensagem final.
* **Trilha Sonora:** Acesso direto à nossa música ("Lisboa") via botão flutuante.
* **Efeitos Visuais:** Animações de corações flutuantes e explosão de confetes ao celebrar nossa data.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica das seções.
* **CSS3:** Design responsivo, animações personalizadas e tipografia via *Google Fonts*.
* **JavaScript (Vanilla):** Lógica do jogo, contador de tempo e transições de tela.
* **Canvas-Confetti:** Biblioteca externa para os efeitos de celebração.

## 📂 Como Rodar o Projeto

1.  Faça o download do arquivo `index.html`.
2.  Abra o arquivo em qualquer navegador (Chrome, Edge, Safari ou Firefox).
3.  **Nota:** É necessário conexão com a internet para carregar as fontes e os efeitos visuais.
